Game of Life Simulator

British mathematician John Horton Conway devised a cellular automaton known as The Game of Life. This is a simulator that runs his game. The game is made up of a 2d grid of cells which interact with their cellular neighbors (up, down, left, right, and diagonally). Each cell follows four rules:

  1. Any live cell with fewer than two live neighbours dies, as if caused by under-population.
  2. Any live cell with two or three live neighbours lives on to the next generation.
  3. Any live cell with more than three live neighbours dies, as if by over-population.
  4. Any dead cell with exactly three live neighbours becomes a live cell, as if by reproduction.

Coin Collector

Random Color Generator

This is my Random Color Generator. It generates random colors based on random values for hue, saturation, and lightness. The user interface has three columns. The first column features random colors among every hue, the second column features warm hues, while the third column features cooler hues.
